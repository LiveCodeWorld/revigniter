Auto-loading Resources

revIgniter comes with an "Auto-load" feature that permits libraries, stacks, helpers, and plugins to be initialized automatically every time the system runs. If you need certain resources globally throughout your application you should consider auto-loading them for convenience.

The following items can be loaded automatically:

  • Core libraries found in the "libraries" folder
  • Stacks found in the "stacks" folder
  • Helper files found in the "helpers" folder
  • Plugins found in the "plugins" folder
  • Custom config files found in the "config" folder
  • Language files found in the "system/language" folder
  • Models found in the "models" folder

To autoload resources, open the application/config/autoload.lc file and add the item you want loaded to the autoload array. You'll find instructions in that file corresponding to each type of item.

Note: Do not include the file extension (.lc) when adding items to the autoload array.
