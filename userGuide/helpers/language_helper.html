Language Helper

The Language Helper file contains handlers that assist in working with language files.

Loading this Helper

This helper is loaded using the following code:

rigLoadHelper "language"

The following handlers are available:

rigLangLine("language line", "element id")

This function returns a line of text from a loaded language file with simplified syntax that may be more desirable for view files than calling rigLangLangLine() in the Language Library. The optional second parameter will also output a form label for you. Example:

put rigLangLine("language_key", "form_item_id") into gData["formLabel"]
-- becomes <label for="form_item_id">language_key</label>
